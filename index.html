Informações básicas de uma página HTML

lang="pt-br"
- Define o idioma da página.
charset="UTF-8"
- É uma tag HTML que define o conjunto de caracteres a ser utilizado na exibição do conteúdo de um site.
<head>
- É usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site.
<title>
- Define o título do documento, mostrado na barra de título de um navegador ou na aba da página.
<link>
- Especifica as relações entre o documento atual e um recurso externo.
Exemplo:
<link:favicon>
- Define o ícone que é exibido no topo da página.
<link:css>
- Define o caminho para o css.

Link

<a>
- Cria referência de link.

- Link de destino.
target="_blank"
- Abrir o link em outra guia.

- Indica que é um link externo.

- Indica que é a proxima página do proprio site.

- Indica que o link volta para página anterior.
rel="nofollow"

CSS para Listas

list-style-type: '\(código do síbolo)' - mudar o síbolo da lista.

list-style-position: inside; - puxar o síbolo para dentro do bloco.

columns: 2; - número de colunas.

Imagem

background-image: url(); - caminho da imagem de fundo.

background-position: left top; - posicionamento da imagem.

background-size: 100px; - tamanho da imagem.

background-repeat: no-repeat; - número de repetições da imagem.

background-size: cover; - preencher o fundo inteiro com a imagem.

background-attachement: fixed; - para a imagem de fundo ficar fixa.

ShortHand:
background: color > image > position > repeat > size(não está funcionando) > attachement;

Iframe

<iframe src="caminho do site"frameborder="0" > - tag do iframe.

target="'name' do iframe" - para abrir um link dentro do Iframe.

sandbox="sandbox" - para bloquear ações dentro do Iframe.

Tabelas

<caption> - título da tabela.

<tr> - linha da tabela.

<th> - título da coluna da tabela.

<td> - dado da tabela.

<thead> - cabeçalho da tabela.

<tbody> - corpo da tabela.

<tfoot> - rodapé da tabela.

border-collapse: collapse; - deixar as linhas da tabela juntas.

vertical-aligh: bottom; - alinhamento vertical.

escope: "col/row"; - para indicar se os dados estão do lado ou abaixo.

colspan="2" - mesclagem de colunas.

rowspan="3" - mesclagem de linhas.

th:nth-child(2n){} - efeito zebrado.

<colgroup> > <col> - selecionar colunas.

Formulários

<form> - criar área de formulário.
action="cadasrto.php" - ligar o formulário a outra linguagem.
method="get/post" - indica como os dados serão enviados.

<label> - liga o input a um texto.
for="(o id do input)"

<input> - cria a área que ira receber os dados.
type="text" - indica o tipo do input.
required - indica que o campo é obrigtório.
minlength="5" - mínimo de caracteres aceitos.
maxlength="15" - maximo de caracteres aceitos.
max="0" - número mínimo aceito.
max="100" - número maximo aceito.
size="30" - o tamanho da caixa.
placeholder="Digite seu email" - dica para o usuário digitar.
step="0.5" - tamanho do salto em caixas numericas.
value="" - valor predefinido para a caixa.
pattern="^\(\d{2}\)\d{4,5}-\d{4}$" - padrão predefinido de escrita.
checked - a caixa já começa marcada.

<textarea> - área de texto.
