Informações básicas de uma página HTML

lang="pt-br" - defini o idioma da página.

charset="UTF-8" - é uma tag HTML que define o conjunto de caracteres a ser utilizado na exibição do conteúdo de um site.

<head> - é usada como cabeçalho, ou seja, um conjunto de informações de uma página de um site.

<title> - define o título do documento, mostrado na barra de título de um navegador ou na aba da página.

<link> - especifica as relações entre o documento atual e um recurso externo.

    Exemplo: 
        <link:favicon> - define o ícone que é exibido no topo da página.
        <link:css> - define o caminho para o css.

Link

<a> - cria referência de link.
href="" - link de destino.
target="_blank" - abrir o link em outra guia.
rel="external" - indica que é um link externo.
rel="next" - indica que é a proxima página do proprio site.
rel="prev" - indica que o link volta para página anterior.
rel="nofollow" - indica que você não sabe a procedência do site em questão.

Download

<a>
href="" - o nome do arquivo que vai ser baixado (tem que está na pasta do projeto)
download="" - o nome do arquivo novamente.
<type="application/pdf"> - o tipo do arquivo.

CSS

Font

font-family: Arial, san-serif; - definir a fonte do texto.

font-weight: (lighter normal bold bolder); - formatação da fonte.

font-style: italic; - estilo da fonte.

font-size: 2.5em; - tamanho da fonte.

ShortHand:
font: font-style font-weight font-size font-family;

Text

text-align: center; - alinhamento do texto.

text-indent: 30px; - identação antes do inicio do paragrafo.

text-decoration: underline; - decoração do texto.

text-shadow: 1px 1px 1px black; - sombra do texto.

text-transform: uppercase; - especifica como capitalizar um texto de um elemento.

color: black; - cor do texto.

line-heigt: 10px; - espaço vertical entre as linhas.

Pseudo-Classe de Links

CSS para Listas

Imagem

background-image: url(); - caminho da imagem de fundo.

background-position: left top; - posicionamento da imagem.

background-size: 100px; - tamanho da imagem.

background-repeat: no-repeat; - número de repetições da imagem.

background-size: cover; - preencher o fundo inteiro com a imagem.

background-attachement: fixed; - para a imagem de fundo ficar fixa.

ShortHand:
background: color > image > position > repeat > size(não está funcionando) > attachement;

Iframe

<iframe src="caminho do site"frameborder="0" > - tag do iframe.

target="'name' do iframe" - para abrir um link dentro do Iframe.

sandbox="sandbox" - para bloquear ações dentro do Iframe.

referrerpolicy="no-referrer" - para não permitir a coleta de dados dentro do Iframe.

Tabelas

<caption> - título da tabela.

<tr> - linha da tabela.

<th> - título da coluna da tabela.

<td> - dado da tabela.

<thead> - cabeçalho da tabela.

<tbody> - corpo da tabela.

<tfoot> - rodapé da tabela.

border-collapse: collapse; - deixar as linhas da tabela juntas.

vertical-aligh: bottom; - alinhamento vertical.

escope: "col/row"; - para indicar se os dados estão do lado ou abaixo.

colspan="2" - mesclagem de colunas.

rowspan="3" - mesclagem de linhas.

th:nth-child(2n){} - efeito zebrado.

<colgroup> > <col> - selecionar colunas.
